# ZineControl Web

A web-based control interface for Nikon cameras, featuring live video streaming and remote settings management.

## Project Structure

- `/frontend`: React + Vite application for the user interface.
- `/backend`: FastAPI server for camera communication (PTP/IP and USB) and video streaming (FFmpeg).

## Quick Start

### 1. Prerequisites
- Python 3.10+
- Node.js & npm
- FFmpeg installed and in your PATH

### 2. Setup Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Or .venv\Scripts\activate on Windows
pip install -r requirements.txt
```

### 3. Setup Frontend
```bash
cd frontend
npm install
```

### 4. Running the App

**Start Backend:**
```bash
cd backend
uvicorn server:app --reload
```

**Start Frontend:**
```bash
cd frontend
npm run dev
```

The app will be available at [http://localhost
:5173](http://localhost:5173).

## Configuration

Environment variables for the backend:
- `NIKON_RTSP_URL`: The RTSP stream URL from your Nikon camera (defaults to `rtsp://127.0.0.1:8554/live`).
- `STREAM_SIZE`: Resolution of the stream (default `1280x720`).
- `STREAM_FPS`: Frames per second (default `30`).
